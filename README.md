# 약초 찾기 모델

### 목적
산에서 자라는 식물 중 일부 독초는 약초와 외형이 유사하여 일반인이 육안으로 구분하기 어렵습니다. <br>
실제로 등산객들이 독초를 약초로 착각하여 섭취하는 사고가 발생하고 있으며, 이러한 문제를 줄이기 위해 이미지 분류 기반 AI 모델을 개발했습니다.<br>
사용자가 식물 이미지를 입력하면 딥러닝 모델이 이미지를 분석하여 약초와 독초를 분류하고 결과를 제공합니다.

### 주요 기능
- 식물 이미지 업로드
- 이미지 전처리 및 AI 모델 추론
- 약초/독초 분류 결과 제공
- 웹 환경에서 모델 결과 확인

### 사용법
1. 구분하고 싶은 식물의 사진을 촬영합니다.
2. 웹 페이지에 이미지를 업로드합니다.
3. AI 모델이 이미지를 분석하여 분류 결과를 제공합니다.

### 개발 환경 및 기술 스택
#### AI Model
- Python
- TensorFlow
- CNN(Convolutional Neural Network) 기반 이미지 분류 모델
#### Dataset
- AI Hub 동의보감 약초 이미지 데이터셋 활용
#### Backend
- Flask

#### 서비스 동작 과정
사용자 이미지 업로드
        ↓
이미지 전처리
        ↓
CNN 기반 분류 모델
        ↓
약초 / 독초 예측
        ↓
Flask 웹 서비스 결과 출력
